Задание 1

1. Инициализируйте проект: выполните команду `npm init -y` и настройте файл `package.json`.

2. Установите зависимости: установите библиотеки `express`, `mongoose` и `dotenv` с помощью команды `npm install express mongoose dotenv`.

3. Создайте структуру проекта: создайте папки `db`, `routes`, `models`.

4. Настройте подключение к MongoDB: создайте файл `db/index.js` и добавьте код для подключения к базе данных.

5. Создайте файл `.env`: в корне проекта создайте файл `.env`, и добавьте в него строку подключения к MongoDB. Пример:

MONGO_URI=mongodb://localhost:27017/project2
PORT=3000

6. Настройте файл `app.js`: создайте этот файл в корне проекта и добавьте код для запуска сервера.
   Задание 2

1. Создайте файл `User.js` в папке `models`.

1. Определите Mongoose-схему для пользователя, как это было показано выше.

1. Добавьте поля `initialBalance`, `currentBalance` и массив транзакций `transactions` в схему пользователя.

1. Создайте и экспортируйте модель `User`, чтобы она была доступна для использования в других частях приложения.
   Задание 3

1. Создайте маршрут `POST /set-balance` в файле `balanceRoutes.js`.

1. Реализуйте валидацию данных: Проверьте, что начальный баланс указан, является числом и не является отрицательным.

1. Создайте пользователя в базе данных с указанным начальным балансом, установите текущий баланс равным начальному и сохраните пустую историю транзакций.

1. Верните успешный ответ с подтверждением создания пользователя и его данными.

1. Обработайте возможные ошибки: В случае ошибки верните соответствующий ответ с сообщением.

transactionSchema: {type ('income', 'expense'). amount, date}
userSchema: { initialBalance, currentBalance, [transactions]}
